# ART RULES (all singles)
# ───────────────────────
# - Square ~512×512 per figure — one subject centered per file
# - Black or transparent background (app adds drop shadow on dark theme)
# - Bold kid-friendly 3D cartoon / clipart — clear outlines, readable colors
# - NO text, labels, or watermarks in the image
# - Left vs right should be obviously comparable (same camera angle when possible)

# FEATURE DEFINITIONS (for prompts + answers)
# ───────────────────────────────────────────
# SHAPE   = same overall outline / silhouette (circle vs star = different shape)
# COLOR   = same main color(s) — close enough that a kid would say "same color"
# TEXTURE = same surface feel (both smooth, both fuzzy, both bumpy, etc.)
#
# MATCHING FEATURES KEY (what kid should select)

# S = Shape matches · C = Color matches · T = Texture matches
